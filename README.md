# Todo List App

A minimalist, no-nonsense task manager built to stay organized without the clutter. This app focuses on clean aesthetics and a logical workflow for managing projects and daily tasks.

### Features
* **Organize by Project**: Separate your work, personal life, and hobbies into distinct lists.
* **Priority Sorting**: Quick visual cues for High, Medium, and Low priority tasks.
* **Persistent Data**: Tasks stay saved in your browser even after a refresh.
* **Safety Checks**: Confirmation prompts before deleting data to prevent accidental clicks.
* **Clean UI**: A high-contrast, monochrome design that keeps the focus on your work.

### Installation & Setup
1. **Clone the repo**
   ```bash
   git clone https://github.com/vbenedeto/todo-list.git
   ```

2. **Install packages**
    ```bash
   npm install
   ```

4. **Start development server**
   ```bash
   npm start
   ```

5. **Production build**
   ```bash
   npm run build
   ```
